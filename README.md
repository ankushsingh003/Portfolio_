# Ankush Singh - ML Engineer Portfolio 🚀

A modern, visually stunning portfolio showcasing cutting-edge Machine Learning, Deep Learning, and GenAI projects. Built with **Django**, **Vanilla CSS**, and **Interactive JavaScript**.

## ✨ Key Features
- **Dynamic Snowfall UI**: Interactive particle-based background animations.
- **3D Tilt Cards**: Smooth interactive hover effects on project showcases.
- **Production-Grade MLOps**: Projects demonstrating end-to-end model development and deployment.
- **Responsive Design**: Seamless experience across mobile, tablet, and desktop.
- **Integrated Contact System**: Fully functional contact form with database persistence.

## 🚀 Local Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/ankushsingh003/portfolio.git
   cd portfolio
   ```

2. **Install Dependencies**:
   ```bash
   pip install django
   ```

3. **Database Migrations**:
   ```bash
   python manage.py migrate
   ```

4. **Run Server**:
   ```bash
   python manage.py runserver
   ```
   Access at `http://127.0.0.1:8000/`.
